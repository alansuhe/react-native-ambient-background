# Security Policy

## Supported Versions

The following versions of react-native-ambient-background are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 0.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please follow these steps:

1. **Do NOT open a public issue** - This could expose the vulnerability to malicious actors.

2. **Email the maintainer directly** at tosuhe@gmail.com with:

   - A description of the vulnerability
   - Steps to reproduce the issue
   - Possible impact
   - Suggested fix (if any)

3. **Allow time for response** - You can expect an initial response within 48 hours.

4. **Coordinated disclosure** - Once the issue is confirmed and fixed, we will:
   - Release a patched version
   - Publish a security advisory
   - Credit you for the discovery (if desired)

## Security Best Practices

When using this library in your project:

- Keep your dependencies up to date
- Review the code before using it in production
- Follow React Native security best practices
- Be cautious with user-provided input (colors, speed values)

## Acknowledgments

We appreciate the security research community's efforts in responsibly disclosing vulnerabilities.

Thank you for helping keep this project and its users safe!
